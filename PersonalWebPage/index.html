¿Cuál es la diferencia entre Internet y la World Wide Web?

Definicion de Internet

Internet es un sistema global que interconecta redes de computadoras en todo el mundo. Conecta los dispositivos mediante cables de cobre, redes inalámbricas o cables de fibra óptica. Utiliza el protocolo TCP / IP para la comunicación de datos. Esta red consta de varios dispositivos, como computadoras, enrutadores, concentradores, conmutadores, repetidores y muchos más. Cada dispositivo en Internet tiene una dirección IP única para identificar el dispositivo.

Definicion de World Wide Web

World Wide Web (WWW) es una aplicación que se ejecuta en Internet. Se refiere a la gran conexión de las páginas web. Estas páginas están enlazadas mediante hipervínculos. Por lo tanto, el usuario puede visitar de una página a otra para acceder fácilmente a la información requerida. Una combinación de páginas web relacionadas se denomina sitio web..

Diferencias

El Internet implementa todas las capas del modelo TCP/IP mientras que el World Wide Web se queda a un nivel mas alto del modelo OSI o TCP/IP.

Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

  • % - Los pocertajes especifican el porcentaje de la propiedad del correspodiente elemento del padre. Ejemplo: 
      .parent {
      width: 400px;
    }
    
    .child {
      width: 50%; /* = 200px */
    }
  • px - Unidad de medida mas utilizada. Este no se correlacionacon al pixel del dispositivo ya que este puede variar entre modelos. Pero se tiene una convencion para que esta unidad se consistencia entre multiples dispositivos
  • pt - punto (1 pulgada = 72 puntos). Medida inutil en el mundo digital ya que proviene de medidas fisicas, pero util al momento de imprimir stylesheets.
Escoger unas medidas u otras dependerá de la necesidad del programador, ya sea para heredar del padre atributos, medidas digitales o medidas para impresiones.

¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

Ya que al visualizar una pagina este tiene que leer y descargar todo el archivo, contando espacios y comentarios. Entre mas espacios y comentarios o texto no relevante del CSS mas tiempo tardará en descargar el archivo y mas lenta será la página.
